# Introduction to SQLAlchemy
###### By Tihon Lvov

This is a non-exhaustive tutorial on use of SQLAlchemy.

Please refer to https://www.sqlalchemy.org/ for more information.

### Instructions:
1. Set up a virtual environment and use pip to install all requirements
2. First review the schema SQL file to understand the database structure
3. Then create a new PSQL database named "test_sqlalchemy"
4. Then review basic_sqlalchemy.py
5. Then review advanced_sqlalchemy.py
